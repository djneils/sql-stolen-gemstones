# SQL Investigation: The Stolen Gemstones

A self-marking SQL mystery activity. A priceless collection of gemstones has been stolen — students must query the database to follow the clues, identify suspects, and crack the case.

## Overview

Browser-based, runs entirely client-side using SQL.js (SQLite in the browser). No installation or server required — just open the HTML file.

## What's Covered

- `SELECT` with `WHERE`, `AND`, `OR`, `NOT`
- `ORDER BY`, `LIMIT`
- `JOIN` — linking evidence, suspects, and locations
- `LIKE` — pattern matching on clue data
- Aggregate functions: `COUNT`, `SUM`, `AVG`
- `GROUP BY` — summarising by category
- Subqueries — nested `SELECT` statements
- `IN`, `BETWEEN`, `IS NULL`

## How to Use

1. Open `stolen-gemstones.html` in any modern web browser
2. Read the scenario briefing and follow each clue
3. Type your SQL into the editor and click **Run** — the activity marks itself automatically
4. Solve all the clues to identify the thief
